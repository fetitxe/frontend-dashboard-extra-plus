Frontend Dashboard Extra Plus WordPress plugin is a supportive plugin for Frontend Dashboard with supportive additional features likes International Telephone,.

= International Telephone =



== Installation ==
1. Upload the “frontend-dashboard-extra-plus” directory to the plugins directory.
2. Go to the plugins setting page and activate “Frontend Dashboard Extra Plus”
3. Go to Frontend Dashboard and add the International Telephone input fields
4. Do save.
